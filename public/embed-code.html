📅 WordPress埋め込みコード

今週の診療カレンダーをWordPressサイトに埋め込むためのコード

方法1: iframeで埋め込み（推奨）

⚠️ 本番URLを取得後、以下のコード内のYOUR_VERCEL_URLを実際のURLに置き換えてください 

<div style="width: 100%; margin: 20px auto;">
  <iframe 
    src="https://YOUR_VERCEL_URL.vercel.app/calendar-embed" 
    width="100%" 
    height="280" 
    frameborder="0" 
    style="border: 1px solid #e0e0e0; border-radius: 8px; box-shadow: 0 2px 8px rgba(0,0,0,0.1);"
    scrolling="no"
    loading="lazy">
  </iframe>
</div>

設定手順

  1. WordPressダッシュボードにログイン
  2. 投稿または固定ページを編集
  3. 「+」ボタンで「カスタムHTML」ブロックを追加
  4. 上記のコードを貼り付け
  5. YOUR_VERCEL_URLを実際のVercelのURLに置き換え
  6. 「公開」ボタンをクリック

サイズ調整

カレンダーの高さを調整したい場合は、height="280"の値を変更してください：

  • 小さいサイズ: height="220"
  • 標準サイズ: height="280"（推奨）
  • 大きいサイズ: height="350"

レスポンシブ対応

モバイル対応のコード（オプション）：


<div style="width: 100%; margin: 20px auto;">
  <div style="position: relative; padding-bottom: 150%; height: 0; overflow: hidden; max-width: 100%;">
    <iframe 
      src="https://YOUR_VERCEL_URL.vercel.app/calendar-embed" 
      style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;"
      frameborder="0" 
      scrolling="no"
      loading="lazy">
    </iframe>
  </div>
</div>
